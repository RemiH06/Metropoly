METROPOLY
Board Game Generator · Guadalajara Metropolitan Area

A Python-powered Monopoly-style board game generator set in the ZMG. Generates print-ready HTML assets — board, property cards, fortune decks and a full rulebook — from a single CSV file.

Features

🗺️
3-Ring Board
Blue (40), yellow (36) and red (32) concentric lanes generated from a config file.
🎨
Auto Color System
13 color groups assigned by price order. No manual configuration needed.
🖼️
Image Scraper
Optional Selenium scraper fetches local photography for each tile automatically.
🎴
Fortune Decks
119 unique fortune cards across 3 lane-specific decks, levels 1–5 balanced.
🏢
12 Companies
Each company has a unique permanent effect — from controlling the jail to taxing all transactions.
📦
gameFactory
One command assembles the complete print-ready game directory with index page.

Quick start

1

Clone & install

git clone https://github.com/RemiH06/Metropoly.git
cd Metropoly
pip install -r requirements.txt
2

Generate everything

python generator.py
3

Assemble print directory

python gameFactory.py

Open juego_completo/indice.html to access all game files.
